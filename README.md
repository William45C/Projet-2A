Commandes :

    Choix des statistiques :
        -"Attaque" Séléctionne la stat attaque
        -"Défense" Séléctionne la stat défense
        -"Agilité" Séléctionne la stat agilité
        -"Ajoute n" n le nombre de point de stat à mettre dans la caractéristique séléctionner
    Combat :
        -"Attaque" Attaque l'ennemi
        -"Fuite" Essaie de fuire
        -"Objets Permet d'accéder aux objets
        -"Inventaire" Permet d'accéder à l'inventaire des armes
    Direction :
        -"Avance" Va tout droit
        -"Droite" Va à droite
        -"Gauche Va à gauche
    Séléction d'objet (Magasin) :
        -"Objet 1" Achete l'objet le plus en haut (numéro 1)
        -"Objet 2" Achete l'objet au mileu (numéro 2)
        -"Objet 3" Achete l'objet du bas (numéro 3)
        -"Partir" Fermer le magasin
    Séléction d'objet (objets) :
        -"Objet n" avec n le numéro de l'objet (à partir de 1)
        -"Partir" fermer les objets
    Coffre :
        -"Ouvrir" Permet d'ouvrir le coffre
        -"Laisser" Permet de ne pas prendre le coffre
    Inventaire des armes :
        -"Inventaire" Ouvre l'inventaire
        -"Objet n" avec n le numéro de l'objet (à partir de 1)
        -"Partir" Ferme l'inventaire 
    Combat de boss :
        -"Attaque" Permet d'attaquer
        -"Esquive" Essaye d'esquiver
        -"Objets" Ouvre les objets
